# Projet Genially 6e

Ce dépôt contient les exercices interactifs, les outils et la documentation du projet Genially destiné aux élèves de 6e.

## Organisation

```text
Sixiemes/
├── README.md
├── AGENTS.md
├── CHANGELOG.md
├── docs/
├── geolib/
├── assets/
├── 01/
├── 02/
├── 03/
├── 04/
├── 05/
└── 06/
```

- `docs/` : règles, décisions et suivi du projet.
- `geolib/` : bibliothèque de géométrie et ses versions.
- `assets/` : ressources communes.
- `01/`, `02/`, etc. : chapitres et exercices.

## Principe de travail

La pédagogie est validée avant le code.

Les exercices restent aussi simples que possible. Toute fonctionnalité utile à plusieurs activités de géométrie est développée dans GeoLib.

## Version stable de GeoLib

La version stable actuelle est indiquée dans `docs/avancement.md`.
