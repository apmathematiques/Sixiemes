# Projet Genially 6e

Ce dépôt contient le développement d'un manuel numérique interactif de mathématiques pour la classe de 6e.

## Organisation

- `docs/` : documentation du projet.
- `geolib/` : bibliothèque de géométrie.
- `01/`, `02/`, ... : chapitres.
- `assets/` : ressources communes.

## Philosophie

La pédagogie est validée avant toute implémentation.

Les exercices restent simples. Les fonctionnalités réutilisables sont intégrées à GeoLib.
