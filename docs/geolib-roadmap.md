# Roadmap GeoLib

## Version stable

GeoLib **1.2**

Fonctionnalités disponibles :

- scène SVG ;
- droites ;
- points ;
- équerre manipulable ;
- rotation de l'équerre ;
- sélection interactive du bord ;
- génération de points atteignables ;
- générateurs aléatoires ;
- vérification d'une construction perpendiculaire.
